# Streamlit heatmap

Небольшое приложение для визуализации таблицы из `daniovision_table.xlsx` в виде heatmap.

## Запуск

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Запуск в Docker

Сборка образа:

```bash
docker build -t heatmap2-streamlit .
```

Запуск контейнера:

```bash
docker run --rm -p 8501:8501 --name heatmap2-app heatmap2-streamlit
```

После запуска приложение будет доступно по адресу `http://localhost:8501`.

## Что умеет

- читает первый лист Excel и восстанавливает merged cells;
- использует названия препаратов как строки и behavioral metrics как столбцы;
- рисует heatmap по `Log2Fold Change`;
- подсвечивает незначимые ячейки серым, если `p-value` больше выбранного порога;
- показывает цветовую полосу классов препаратов слева;
- даёт фильтры по классам, метрикам и порогу значимости.

## Структура входных данных

Ожидается таблица, похожая на текущий файл:

- колонка `A` — класс препарата;
- колонка `B` — название препарата;
- в первой строке — названия метрик;
- во второй строке под каждой метрикой идут `Log2Fold Change` и `p-value`.

Если структура будет такой же, приложение сможет читать и другие похожие `.xlsx`.
